# elevator-controller-vlsi
A Verilog-based Elevator Controller using Finite State Machine (FSM) logic to manage floor movement, door control, and direction efficiently implemented using Cadence Xcelium and Virtuoso.

Features

upto 128 floor handling

Automatic up/down control

Door open/close logic

Files
elevator_controller.v      # Main code  
elevator_controller_tb.v   # Testbench
